### Tech Stack:

### Install and Run:

- Run 'npm start'
## If doesn't work with 'npm start' then:
- Remove node_modules folder
- Remove package-lock.json file
- Run 'npm install'
- Run 'npm i -S react-scripts'
- Run 'npm start'
